## ğŸ²&nbsp; About

This Android application is used for api data management used for the RPG Aux App <a href="https://github.com/edvaldinhs/borderK">borderK</a> project. Communication is implemented through periodic state polling from a custom API, with the user issuing pings/signals to modify the current state and game_id.
